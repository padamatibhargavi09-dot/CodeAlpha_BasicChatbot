# PyBot - Basic Chatbot 🤖

## About
A rule-based chatbot built in pure Python with a Tkinter GUI and on-screen keyboard.
Built as Task 4 for CodeAlpha Internship.

## Features
- On-screen clickable keyboard
- Quick phrase buttons
- Dark themed chat interface
- Typing indicator
- Timestamps on messages

## How to Run
python pybot.py

## Topics Covered
- if-elif logic
- Functions
- Loops
- Input/Output
- Tkinter GUI

## Tech Used
- Python 3.6+
- Tkinter (built-in)
- No external libraries needed!
